# 190121~190122 Bootstrap

## 1. Utilities

### 1.1 Spacing

* `m-0` : margin: 0
* `mr-0` : margin-right: 0
* `mx-0` : margin-right, margin-left: 0 (오른쪽, 왼쪽 똑같이 적용됨.)
* `mx-auto` : margin 남은 여백 나눠주기.
* `pt-0` : padding-top: 0
* `pb-0` : padding-bottom: 0
* `pr-0` : padding-right: 0
