# Toy Project1 명세서 

## 1. 목표

* Django를 이용하여 SSAFY 대전 1반 동기들과 함께 학습 사이트를 공유할 수 있는 Web Site 구축하기
  * 다양한 교육을 받기 때문에 카테고리 별로 학습 사이트를 구분하여 정보를 제공한다.



## 2. 준비 사항

1. Django
2. Materialize



## 3. 기능 설계

1. User 관리
   * 회원가입, 로그인, 로그아웃
   * 회원가입 시 관심있는 분야 선택하기
   * 선택한 분야를 먼저 Main에 보여주기



2. 카테고리별 정보 제공

   > card 
   >
   > * Title
   > * Description
   > * Button 생성하여 Link이동

   > 좋아요기능을 추가하여 좋아요 순으로 나열하기

   * 강의
   * Django 
   * Web
   * Front-end
   * Back-end
   * 알고리즘
   * Python - 문제풀이
   * C언어
   * AI
   * Q&A



3. 누구든지 추가 가능하도록 만들기
