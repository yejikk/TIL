# git - pull request

1. `git checkout -b yeji`
   * branch 생성 및 branch로 실행

2. `git status` 
   * 현재 디렉토리 상태 확인하기

3. `git add .`
   * 파일 추가

4. `git commit -m " "`
   * commit 

5. `git push origin yeji`
   * 내 branch로 push

6.  pull-request 후 merge
7. merge 완료 후 `git pull origin master`
   * pull은 무조건 master로 해주기
